# Day 2 Interview Questions:-
##Explain OSI Model?
  Answer: It's a universal 7 layer frame work. It standadizes hoe computers communicates over internet and helps engineers to troubleshoot.
  #Application Layer:
      The top layer directly interacting with The software.Protocols here allows users top access network services. (HTTP for Web browser,SMTP for emails)
  #Presentation Layer:
      It Translets,encrytpes and compress the data so the reciver can accuretly read it. (SSL/TLS encryption, image)
  #Session Layer:
      It establishes, maintains and terminates the communication sessions between applications.
  #Transport Layer:
      Breaks data into segments, ensuring end-to-end relability,flow control and error recovery.(using protocols like TCP/IP or UDP)
  #Network Layer:
      Handles the logical routing,forwading, assigning source and destination IP address to determine the best path accross different networks.(Router) 
  #Data Link Layer:
      Package raw data into frames and ensures error free node to node delivery accross a local network using physical MAC Address.(Switches)
  #Physical Layer:
      The foundation, it manages the transmission of raw bit streams over physical cables, wire signals  or fiber optics cables.(Hubs, cables)
